    Mulai Tutorial
    1
    Masuk ke IP Modem

    Buka Chrome dan ketik: 192.168.1.1

    2
    Login Admin
    • Username: superadmin
    • Password: suportadmin
    3
    Buka Menu Network

    Klik menu Network di dashboard modem.

    4
    Pilih Menu WLAN

    Klik WLAN untuk setting WiFi.

    5
    Ganti Nama & Password
    • SSID Name → Nama WiFi
    • WPA Passphrase → Password WiFi
    • Jangan lupa klik Submit setelah diganti.
    Tips WiFi Anti Ribet
    • Password minimal 8 karakter
    • Campur huruf & angka
    • Jangan kasih ke sembarang orang
